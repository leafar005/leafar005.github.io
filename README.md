# Portfolio

Portfolio personal para leafar005 (yo), alojado en GitHub Pages, que sale mas barato

## Sobre el proyecto

Este proyecto es una página web interactiva fuertemente inspirada en la interfaz y el HUD del videojuego **Signalis** (estas obligado a jugarlo si lees esto, no sabes lo que te pierdes).

Características principales:
- Interfaz inmersiva con efectos de pantalla CRT, scanlines y viñeta.
- Sistema de inventario interactivo.
- Integración de visor de modelos 3D (`model-viewer`).
- Diseño responsivo adaptado para PC y dispositivos móviles.

## Tecnologías utilizadas

- **HTML5** y **CSS3** (animaciones, variables, Grid/Flexbox).
- **JavaScript** (Vanilla JS para la lógica interactiva).
- [Model-viewer](https://modelviewer.dev/) para la renderización de elementos 3D.
